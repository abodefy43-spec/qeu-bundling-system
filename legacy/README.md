# legacy

Archived implementation assets from the pre-reset repository live here.

This directory is intentionally not part of the active Phase 0 runtime.

- `config/` keeps the old Dockerfiles and split dependency files
- `static/` and `templates/` keep the retired dashboard assets
- `src/legacy/qeu_bundling` keeps the previous monolithic application code
- `tests/legacy/` keeps the previous test suite for reference only
