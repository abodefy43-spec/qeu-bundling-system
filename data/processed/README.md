# data/processed

Pipeline-generated intermediate artifacts, for example:

- `filtered_orders.pkl`
- `product_embeddings.npy`
- `embedding_mapping.json`
- `copurchase_scores.csv`
- `product_categories.csv`
- `product_recipe_scores.csv`
- `top_bundles.csv`

These files can be regenerated by running the pipeline.

Sensitive/derived artifacts from real customer data are intentionally excluded from Git history.
